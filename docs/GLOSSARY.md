# Glossary

## Civic Issue

A public-space problem that a person may report to a municipal team, campus facilities team, building manager, NGO, neighborhood group, or service team.

## Report

The generated copy-ready message that a user can paste into a service portal, email, chat, help desk, or contact form.

## Evidence

Optional context that makes a report easier to verify, such as a photo, observation time, repeated pattern, or nearby landmark.

## Offline-First

The app is useful without a backend or account. The first version does not send user-entered text to any server.
