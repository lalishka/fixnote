# Setup

## Requirements

- A modern browser.
- Node.js for tests.
- Python 3 only if using the local static server command.

No npm dependencies are required.

## Run

Open `index.html` directly, or serve the folder:

```bash
npm run dev
```

Then open:

```text
http://localhost:4173
```

## Test

```bash
npm test
```

The tests use Node's built-in `node:test` runner.
